Dunhuang Pattern Visualization / 敦煌纹样可视化系统

一个面向敦煌纹样数据的交互式前端可视化网页，用于纹样浏览、筛选、分析与展示。

Watch Demo

Context

把研究型文化遗产数据转化成可浏览、可分析的前端产品

该项目围绕敦煌藻井与纹样图像数据，设计了从封面叙事、全局统计、图库筛选到单图详情分析的完整浏览路径。页面侧重图像内容展示、色彩统计、纹样共现、聚类浏览和详情页交互，适合作为前端工程能力、可视化表达和产品化思考的综合展示案例。

Demo

完整高清视频

视频展示首页动画、进入数据库、图库筛选、纹样详情、统计图表、聚类可视化和主要页面交互。

Implemented Features

实现功能

沉浸式入口与数据库导航

从敦煌主题封面进入全局统计和图案库，用动画与视觉入口串联浏览路径。

纹样图库与多维筛选

支持按朝代、位置等维度筛选图像，并从图库跳转到单张纹样详情。

单图详情与纹样定位

在详情页展示图像元信息、典型纹样、局部区域、布局层级和相关纹样信息。

统计与可视化分析

实现纹样计数、共现矩阵、网络图、朝代色彩演变、颜色聚类和相似纹样展示。

Role

  • 实现纹样数据浏览、筛选、分类、图像展示、详情页交互和可视化图表呈现。
