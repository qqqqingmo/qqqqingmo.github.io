Project

Dunhuang Pattern Visualization

敦煌纹样可视化系统

面向敦煌藻井与纹样图像数据的交互式前端可视化网页，将图像浏览、纹样筛选、详情分析和统计图表组织成一条连续的研究数据探索路径。

Implementation

Implemented modules

01

入口叙事与信息架构

从敦煌主题封面进入数据库，把视觉叙事和数据浏览连接起来，避免 demo 只停留在静态展示。

  • 封面入口：用主题化视觉和动效建立项目语境。
  • 路径组织：串联全局统计、图案库和详情页，形成可回溯的浏览流程。
  • 导航逻辑：把研究数据拆成 overview、gallery、detail、analytics 等可理解层级。
02

纹样图库与多维筛选

将纹样图像组织成可筛选、可比较的图库界面，让用户能从大规模图像集合中定位感兴趣的样本。

  • 筛选维度：支持按朝代、位置、图像类别等条件缩小结果范围。
  • 结果浏览：以图像为中心呈现样本，同时保留必要的元信息。
  • 详情跳转：从图库直接进入单张纹样详情，保持探索链路连续。
03

单图详情与纹样定位

详情页承担从“看见图像”到“理解图像结构”的过渡，突出纹样、局部区域和布局关系。

  • 图像元信息：展示编号、来源、年代、位置等基础字段。
  • 局部定位：呈现典型纹样、局部区域和相关标注，帮助理解纹样在图像中的位置。
  • 关联信息：补充布局层级、相似纹样与相关样本，支持继续探索。
04

统计图表与视觉分析

将纹样数据进一步转化为可分析的图表视图，用于观察数量分布、共现关系和颜色变化。

  • 纹样统计：实现计数、占比和类别分布等基础统计视图。
  • 关系分析：通过共现矩阵和网络图展示纹样之间的关联结构。
  • 颜色分析：呈现朝代色彩演变、颜色聚类和相似纹样检索结果。
