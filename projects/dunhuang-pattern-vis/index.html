Project

Dunhuang Pattern Visualization

敦煌纹样可视化系统

面向敦煌藻井与纹样图像数据的交互式前端可视化网页，将图像浏览、纹样筛选、详情分析和统计图表组织成一条连续的研究数据探索路径。

Type
Research Visualization
Focus
Cultural heritage data, Visual analytics, Frontend interaction
My role
Frontend system, Data preparation, Visual analytics, UI design

Demo

System walkthrough

系统涵盖从封面动画、全局统计、图库筛选到单图详情的完整交互路径，提供计数统计、共现矩阵、网络图、颜色聚类等多种分析视图；支持按朝代、位置等维度筛选，可查看单张图像的元信息、典型纹样、局部区域、色彩分析、所在洞窟以及相关样本。

Role

Contribution

Frontend system

基本独立完成前端页面设计与实现，搭建从封面、全局统计、图库到详情页的完整交互流程。

  • 设计页面结构、路由关系和主要交互状态。
  • 实现图像浏览、条件筛选、详情页切换和图表展示。
  • 整理作品展示版本，使研究型项目可以作为可访问的网页 demo 呈现。

Data and analytics

完成数据处理、统计分析和后端侧统计代码，将原始研究数据整理为前端清晰的数据结构。

  • 分类纹样、图像元信息和整理统计字段。
  • 实现纹样计数、共现关系、颜色聚类等分析结果。
  • 分析输出并映射到所需的数据格式。

Visualization design

负责完整UI设计，把文化遗产数据转译为更易阅读的界面表达，在学术信息密度和网页可用性之间平衡。

  • 根据研究问题选择图库、矩阵、网络图和聚类视图等表达形式。
  • 控制页面叙事节奏，使浏览路径从整体概览逐步进入局部分析。
